Browser Rendering Pipeline

HTML
↓
Parsing
↓
Tokenization
↓
DOM Tree
+
CSS
↓
CSSOM Tree
↓
Render Tree

Parsing

The browser reads the HTML document and converts it into tokens.

Tokenization

HTML is broken into meaningful tokens like tags, attributes and text.

DOM Tree

Tokens are converted into nodes forming the Document Object Model.

CSSOM Tree

CSS rules are parsed and converted into a CSS Object Model.

Render Tree

DOM Tree and CSSOM Tree combine to create the Render Tree used for painting.
